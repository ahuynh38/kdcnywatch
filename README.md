# Overwatch Dashboard

A personal dashboard for tracking Overwatch stats across my friend group. Displays player data in a clean, shareable interface with the ability to manually refresh stats on demand and add new players to the tracking list over time.

## What It Does

- Displays current Overwatch stats for a configured list of players
- Allows manual stat refreshes via a button on the dashboard
- Supports adding new players to the tracking list from the dashboard
- Designed to be shareable — friends can view the dashboard without any login
